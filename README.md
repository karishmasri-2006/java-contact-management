##  Description
A Java console-based Contact Management System developed using OOP concepts, ArrayList, and File Handling. It allows users to add, view, search, update, and delete contact records. The system also saves contacts to a text file and automatically loads them when the program starts, ensuring data persistence.

##  Features
- Add Contact
- View All Contacts
- Search Contact by Name
- Update Existing Contact
- Delete Contact
- Save Contacts to `contacts.txt`
- Auto-load Contacts on Program Start

##  Technologies Used
- **Java** – Core Programming Language
- **OOP Concepts** – Classes, Objects, Encapsulation
- **ArrayList** – Dynamic data storage using the Collections Framework
- **File Handling** – `FileWriter`, `FileReader`, `BufferedReader`
- **Collections Framework** – Efficient data management

##  How to Run

1. Navigate to the project directory:

```bash
cd java-contact-management
```

2. Compile all Java files:

```bash
javac *.java
```

3. Run the program:

```bash
java Main
```
